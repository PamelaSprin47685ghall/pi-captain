# JSON Canvas Examples

Committed reference canvases used to calibrate visual quality when generating new canvases.

## pi-captain-showcase.canvas

A real-world showcase canvas for the pi-captain repository.
Open in Obsidian to see what a well-formed canvas should look like.

**Stats**: 37 nodes (7 groups + 30 text), 9 edges, ~3050×2530 px total

Use this as the **visual quality bar** when generating canvases:
> "Does this look like the showcase?" means: readable text, correct group containment,
> labelled edges, semantic colors, nodes big enough to show full content without clipping.
